Evaluation¶

1. Time for ECDF read, invertion and write cycle. Table and counter size is varied.¶

  • All combinations of:
    • m = {8,9,10,11,12,13}, represents table size.
    • c = {1024,2048,4096,8192,16384,32768}, represents counter size.
  • For each combination do two variations:
    • Variation 1. Run for 10 seconds. Control Plane goes through the entirety of ECDF invertion cycle.
    • Variation 2. Run for 10 seconds. Control Plane skips everything except writing to the Data Plane (dummy values).
  • Additionally:
    • Run traffic, 1 gold flow and 1 silver flow.
    • Set ECDF update timer to 0.
    • α is set to 0.3125 and β is set to 3.125.

(tests ending with _2 has the time for counter pull and write dummy values) (Try to draw all Ev1_m8 ... Ev1_m13 in the same graph)

    • m represents table size.
    • c represents counter size.
    • The ECDF update timer in the Control Plane is set to 0.
    • α is set to 0.3125 and β is set to 3.125, unless otherwise specified.
